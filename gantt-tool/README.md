# gantt-tool

一个用于多项目月度计划录入、文本抽取、自动计算和每项目甘特图展示的小工具，支持本地模式和后端 SQLite 模式。

## 已实现功能

1. 多项目管理：每个项目独立数据、独立甘特图。
2. 项目基础信息：覆盖任务名称、部门、责任人、立项/验收日期等字段。
3. 文本抽取：从输入文本中提取常见字段（可人工校正）。
4. 月度明细表：支持新增/编辑/删除任务明细。
5. 自动计算：
   - 周期（天）= 预计完成日期 - 预计开始日期 + 1
   - 任务有实际完成日期时自动将进度回填为 100（可再改）
   - 整体执行百分比按周期加权计算
6. 甘特图：按当前项目渲染，支持按月份过滤，支持延期高亮、实际完成标记。
7. 多项目汇总看板：项目总数、延期项目、任务总数、平均进度、部门统计、延期任务 TOP 10。
8. 数据持久化：
   - 本地模式：浏览器 localStorage
   - 后端模式：SQLite（通过 API 自动保存）
9. 数据导入导出：JSON 文件。

## 使用方式

### 方式一：本地模式（最快）

1. 直接用浏览器打开 `index.html`。
2. 左侧新建项目，填写项目基础信息。
3. 在“文本抽取”区域粘贴描述，点击“从文本提取”。
4. 在“工作计划明细”中维护每月任务。
5. 在“项目甘特图”查看当前项目进度。

### 方式二：后端数据库模式（推荐）

1. 在工作区根目录启动后端：

```bash
python backend/server.py
```

2. 浏览器访问：

```text
http://127.0.0.1:8080
```

3. 右上角会显示“存储模式：后端 SQLite”。

说明：
1. 后端会自动创建数据库文件 `backend/data.db`。
2. 前端每次改动会自动写入数据库，同时保留一份浏览器本地缓存。
